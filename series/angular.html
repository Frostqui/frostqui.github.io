Desde cómo instalar Angular hasta conceptos avanzados como testing unitarios serie: angular excerpt: Curso completo de Angular, uno de los frameworks Typescript más populares para el desarrollo web. No confundir con AngularJS, en esta serie de artículos usaremos a partir de Angular 2. --- Aprenderás a usar Angular, uno de los mejores frameworks para el desarrollo web. Si hacemos una comparación de Angular vs React, Angular añade una serie de ventajas al desarrollo, cómo por ejemplo, inyección de dependencias, tipado y un modelo MVVM (Model-View-ViewModel). En este curso veremos cómo hacer una web al completo con Angular, empezaremos desde la base, cómo instalarlo, cómo empezar a usarlo y cómo se la sintaxis de Typescript, y terminaremos viendo cómo hacer testing, pipes, cómo crear un sistema de login y register, etc.
