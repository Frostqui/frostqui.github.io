¿Quieres aprender Angular 6 gratis?

Bienvenido al curso online gratis y en español de angular, uno de los mejores frameworks para la creación de página web. Este framework está apoyado por Google y cuenta con una comunidad enorme. Uno de los puntos fuertes de Angular es si arquitectura MVC (Modelo Vista Controlador) y su inyección de dependencias gracias a su sintaxis, Typescript.

En este curso de Angular lo abordaremos desde un nivel básico y pasaremos por muchas de las cosas que puede ofrecer este framework. Para finalizar daremos pinceladas de conceptos mucho más técnicos como testing y despliegue, aunque no te asustes porque no se trata de un curso avanzado 100%.

Accede GRATIS al curso

Requisitos recomendados

  • Saber algo de HTML y CSS
  • Saber lo básico de algún lenguaje de programación como Javascript o Java (sobre todo Java porque se parece a Typescript)
  • Saber hacer orientación a objetos
  • ¡Y listo! No necesitas experiencia en AngularJS (la versión anterior de Angular) ni en otro framework Javascript como React

¿Para quién es este curso?

Angular desde 0 es un curso orientado a todo tipo de personas, como he dicho antes vas a necesitar saber lo básico de HMTL Y CSS y mucho mejor si has usado alguna vez Javascript. Si tienes un perfil back, este curso de Angular también es recomendado ya que la sintaxis de Angular no es muy distinta de lenguajes del estilo de Java.
